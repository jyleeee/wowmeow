新媒体运营

New media operation

工作内容:

1.负责微信官方账号的运营,日常内容编辑、发布、维护、管理、互动、提高影响力和关注度；
2.负责微信公众账号推广模式与渠道的探索，了解用户需求，收集用户反馈，分析用户行为；
3.负责建立与外部各渠道的良好关系，整合各渠道资源，并开发新的传播形式新媒介开发；
4.利用微信平台推广企业的其他业务或服务的品牌和产品；
5.跟踪微信推广效果,分析数据并反馈,总结经验,建立有效运营手段提升用户活跃度,增加粉丝数量；
6.利用社会化新媒体沟通方式：微博、微信、论坛进行社会化营销，获取用户资源，创造口碑；
7.关注并搜集竞争对手等相关活动，协助制定相关的营销策略；
8.具体执行其他微媒体传播信息编撰、发布、互动、主动搜索、报表汇总等工作；
9.跟进公司情况策划并执行热门话题营销，并实施线上及线下活动。

要求:

1.两年以上的互联网从业经验；
2.熟悉新媒体运营的工作内容；
3.喜欢宠物，男生优先考虑；
4.有扎实的文案功底。

Web前端工程师

Web front-end engineers

工作内容:

1.负责Web页面前端开发，编写HTML、CSS和JS代码，配合后端工程师完成系统开发；
2.对公司产品页面持续进行前端优化，改善用户访问体验；
3.制作或协助制作各种页面组件或页面模版；
4.制定公司JS、CSS、HTML开发规范并推进执行；
5.进行前端基础框架开发，与其他研发人员进行充分的沟通与协作；

要求:

1.大专及以上学历，计算机相关专业毕业，至少相关工作2年以上；有移动端的开发经验优先；
2.精通JS面向对象程序设计以主流JS类库，具有较强的前端代码开发能力满足产品的UI交互设计需求；
3.对web前端脚本、应用程序的性能优化有较深的理解，并有相关的实践经验；
4.熟悉各主流浏览器的差异能很好的解决浏览器的问题；
5.较强的团队协作能力与沟通能力，并与设计人员，程序开发人员进行较好的配合。

数据分析专员

Data Analysis

工作内容:

1.充分熟悉和理解运营需求，根据业务实际需要制定数据需求方案，并推动执行形成价值输出点；
2.从数据角度对于运营活动、产品优化给予建议并协调各方资源，推动数据监控功能的优化和完善；
3.深入挖掘用户属性及行为数据，从数据的角度发现问题、及发掘业务优化与提升机会，推动运营；
4.建立数据分析模板，针对用户行为渠道等进行监控和统计；
5.根据实际数据，分析用户及市场特征，提供相应的运营建议，和用户的黏性；
6.整理编写数据分析报告，及时发现和分析其中隐含的变化和问题，为业务发展提供决策支持。

要求:

1.一年以上数据统计分析、数据挖掘等相关工作经验，有互联网行业数据分析经验者优先；
2.熟悉数据建模知识、数据挖掘理论，掌握数据分析体系方法；
3.强烈的责任感和很好的团队合作精神，能够承受较强的工作压力；
4.具有优秀的沟通能力、执行力和产品判断力。
